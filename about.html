Om denna webbplats

labb 2 i Backend-baserad webbutveckling.

På webbplatsen kan man lägga till arbetserfarenheter som sedan visas i en lista. Det sker genom CRUD och fetch anrop

Syfte

Visa och hantera arbetslivserfarenheter i ett CV.

Teknik som används

Node.js, Express, Mongoose (tidigare better-sqlite3)

HTML, CSS och JS med Fetch API

Slutsatser

I denna uppgift har jag lärt mig hur man kopplar ihop frontend och backend med REST API, hur viktigt det är med validering, samt hur man använder Fetch för CRUD.
